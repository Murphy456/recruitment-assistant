# 招聘助手 - 安装使用指南

## 一、环境准备

### 1. 检查 Node.js

打开命令行（按 `Win + R`，输入 `cmd`，回车），输入：

```bash
node -v
```

- 如果显示版本号（如 `v20.12.0`），说明已安装，继续下一步
- 如果显示"不是内部或外部命令"，需要安装 Node.js

### 2. 安装 Node.js（如已安装可跳过）

1. 访问 https://nodejs.org/
2. 下载 **LTS 版本**（长期支持版）
3. 双击安装包，一路点击"下一步"完成安装
4. 重新打开命令行，输入 `node -v` 验证安装成功

---

## 二、安装扩展

### 方式一：直接安装（推荐）

1. 解压 `recruitment-assistant-extension.zip` 文件
2. 打开 Chrome 浏览器
3. 在地址栏输入：`chrome://extensions/`
4. 打开右上角的 **"开发者模式"** 开关
5. 点击 **"加载已解压的扩展程序"**
6. 选择解压后的 `dist` 文件夹
7. 看到扩展图标出现在浏览器右上角，安装成功！

### 方式二：从源码构建

1. 解压源码包到任意目录
2. 打开命令行，进入该目录：
   ```bash
   cd recruitment-assistant
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 构建扩展：
   ```bash
   npm run build
   ```
5. 按"方式一"步骤加载 `dist` 文件夹

## 三、配置 AI 服务

### 1. 打开设置

- 点击浏览器右上角的扩展图标
- 点击 **"设置"** 按钮

### 2. 配置 AI 提供商

支持的 AI 服务（任选一个）：

| 提供商 | 获取 API Key 地址 |
|--------|-------------------|
| 通义千问（推荐） | https://dashscope.console.aliyun.com/ |
| OpenAI | https://platform.openai.com/api-keys |
| 智谱 AI | https://open.bigmodel.cn/ |
| Moonshot | https://platform.moonshot.cn/ |

### 3. 填写配置

1. 选择 **提供商**
2. 填写 **模型名称**（通义千问推荐：`qwen-plus`）
3. 填写 **API Key**
4. 配置会自动保存

## 四、创建职位画像

### 1. 进入设置页面

点击扩展图标 → 设置

### 2. 添加职位画像

填写以下信息：
- **职位名称**：如"前端工程师"
- **职位描述**：职位详情
- **学历要求**：本科/硕士等
- **经验要求**：工作年限
- **必备技能**：用逗号分隔，如"React, TypeScript, Node.js"
- **消息模板**：招呼语，可用变量：
  - `{name}` - 候选人姓名
  - `{position}` - 应聘职位
  - `{company}` - 公司名称
  - `{education}` - 学历
  - `{experience}` - 工作年限

示例模板：
```
您好{name}，我是{company}的HR，看到您的简历非常匹配我们的{position}岗位，期待与您进一步沟通！
```

## 五、使用扩展

### 1. 访问招聘平台

目前支持：
- **智联招聘**：https://www.zhaopin.com/

### 2. 打开侧边栏

- 进入简历列表页面
- 点击扩展图标
- 点击 **"打开侧边栏"**
- 侧边栏会显示在页面右侧

### 3. 筛选候选人

1. 在侧边栏选择 **职位画像**
2. 点击 **"批量分析"** 按钮
3. 等待 AI 分析完成
4. 查看匹配分数和推荐理由

### 4. 发送招呼

1. 点击候选人卡片查看详情
2. 确认匹配信息
3. 点击 **"发送招呼"** 按钮

## 六、常见问题

### Q: 扩展图标不显示？

1. 点击 Chrome 右上角的拼图图标
2. 找到"招聘助手"
3. 点击图钉固定到工具栏

### Q: 侧边栏不显示？

1. 刷新招聘平台页面
2. 确保当前页面是 `zhaopin.com`
3. 重新点击扩展图标

### Q: AI 分析失败？

1. 检查 API Key 是否正确
2. 检查网络连接
3. 检查 API 余额是否充足

### Q: 发送招呼失败？

1. 检查是否达到每日发送上限
2. 检查消息模板是否设置
3. 刷新页面重试

## 七、安全说明

- API Key 仅存储在本地浏览器中
- 简历数据不会上传到第三方服务器
- 建议定期更换 API Key
