# 招聘助手 v1.0.0

## 快速开始

### Windows 用户

1. 双击运行 `check-env.bat` 检查环境
2. 双击运行 `build.bat` 构建并打包
3. 解压 `release/recruitment-assistant-extension.zip`
4. 在 Chrome 中加载扩展

### 手动安装

详见 `INSTALLATION.md` 文件

## 文件说明

| 文件 | 说明 |
|------|------|
| `INSTALLATION.md` | 详细安装使用指南 |
| `check-env.bat` | 环境检查脚本 |
| `build.bat` | 构建打包脚本 |
| `release/` | 输出目录，包含扩展压缩包 |

## 系统要求

- Node.js 18+
- Chrome 浏览器
- Windows 10/11

## 功能特性

- ✅ 智能简历匹配分析
- ✅ AI 驱动的候选人评估
- ✅ 一键发送招呼消息
- ✅ 支持通义千问、OpenAI 等多种 AI 服务
- ✅ 本地数据存储，保护隐私
